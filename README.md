# Compliance Advisor Dashboard

A standalone compliance management dashboard with demo data mode. Built with vanilla HTML, CSS, and JavaScript.

## Quick Start

```bash
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080) in your browser. The dashboard loads with demo data automatically when no API is configured.

You can also open `index.html` directly in a browser.
